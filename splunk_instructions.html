What is Splunk?

Splunk is a fully featured, powerful platform for collecting, searching, monitoring and analyzing machine data. When paired with Suricata, Splunk is used to collect, search and analyze alert data produced by Suricata.

Splunk can be made to understand Suricata log data with the addition of a “Snort for Splunk” add on.

Splunk runs on an Intel host and provides a web browser based user interface. Several views of Suricata data as visualized with Splunk follow.





Installing Splunk

Splunk is a commercial application. It can be gotten at the Splunk web site.

Splunk can be installed on the host containing a TileEncore-GX.

If you are working with either a TilEmpower-GX and TilExtreme-GX platform Splunk is installed on another host and will connect to Splunk on the Tilera platform via the network.

To download Splunk visit the Splunk web site.

And select FREE DOWNLOAD

After completing your registration and download you'll be at a page containing Splunk installation instructions.

Follow the provided instructions to install Splunk on your platform.

The next step is to install the Snort for Splunk app to your Splunk installation.

The simplest way to do this is from within Splunk select the “App” pulldown menu and select “Find more apps...”
